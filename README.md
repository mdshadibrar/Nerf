## Project Overview

Tiny NeRF is a lightweight TensorFlow implementation of Neural Radiance Fields (NeRF) for novel view synthesis.

### Features
- Positional Encoding
- MLP-based Neural Rendering
- Volume Rendering
- Ray Sampling
- 360° Scene Rendering
